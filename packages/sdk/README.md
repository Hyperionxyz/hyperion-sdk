# @hyperionxyz/sdk

This SDK allows you to interact with the Hyperion API.  
You can use it to request data, create pools/positions and more.

## Installation

```bash
pnpm install @aptos-labs/ts-sdk @aptos-labs/script-composer-sdk @hyperionxyz/sdk
```
